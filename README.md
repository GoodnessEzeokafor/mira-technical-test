## Setup Project

1. Clone Repository:

```

git clone -b main https://github.com/GoodnessEzeokafor/mira-technical-test.git

```

2. Install Dependencies:

```

npm install

```

3. Set Up Database:

```

npm run db:migrate

```

4. Configure Environment Variables:

```

cp .env.sample > .env

```

5. Start Development Server:

```

npm run start:dev

```
